# 🔸 2. Weights and Bias

---

## 🔹 What are Weights?

👉 Importance given to each input

---

## 🔹 What is Bias?

👉 Extra adjustment to output

---

### Example:

* Input word importance varies
* Model adjusts importance using weights

---

## 🔹 Intuition

👉 Like giving marks:

* Important features → higher weight
* Less important → lower weight

---

# 🔸 4. Forward Pass

---

## 🔹 What is Forward Pass?

👉 Data flows from input → output

---

### Steps:

1. Input enters
2. Multiply by weights
3. Add bias
4. Apply activation
5. Generate output

---

## 🔹 Example

Text → embedding → prediction

---

# 🔸 5. Backpropagation (Learning Process)

---

## 🔹 What is Backpropagation?

👉 Process of correcting errors

---

## 🔹 Steps:

1. Compute error (loss)
2. Send error backward
3. Adjust weights
