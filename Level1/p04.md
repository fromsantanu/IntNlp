# 📘 Chapter 1.4 — Basic Linguistics for NLP

---

To understand language properly, NLP borrows ideas from **linguistics** — the scientific study of language.

👉 Think of linguistics as the **rulebook of language**, and NLP uses these rules to help computers understand text.

---

## 🔹 Why Linguistics is Important in NLP

When humans read a sentence, we naturally understand:

* Words
* Grammar
* Meaning
* Context

👉 But computers don’t have this natural ability.

So, NLP uses linguistics to break language into **structured levels**, making it easier for machines to process.

---

## 🔹 Main Levels of Linguistics in NLP

There are **five key levels**:

1. Morphology
2. Syntax
3. Semantics
4. Pragmatics
5. Phonology (less used in text NLP, more in speech)

Let’s understand each in a simple way.

---

## 🔸 1. Morphology (Structure of Words)

Morphology studies how words are formed.

### Example:

* “unhappiness”
  → un + happy + ness

👉 Here:

* **un** = prefix
* **happy** = root word
* **ness** = suffix

👉 NLP uses this to:

* understand word variations
* perform stemming and lemmatization

---

## 🔸 2. Syntax (Sentence Structure)

Syntax deals with **grammar and sentence structure**.

![Image](https://images.openai.com/static-rsc-4/k3WFKhENrTNUdxTmu0UVDvpFm4_XzjSgJucsKqC73IqRtUCZsMqDKi1LXnPXedJbolv5AQaFvRiPR1eG49UAYZWYxRRXm0QlnbcKdQznwF7Xu0LTCt6SoCDGooJs-ws5giCP-ePi8TUgW_4Chd-ntKlm5SwIt25liUuhNVCQ6GnpuXslf8aT_3mFpBu0-Gz-?purpose=fullsize)

### Example:

* “The cat eats fish”

👉 Structure:

* Subject → The cat
* Verb → eats
* Object → fish

👉 NLP uses syntax to:

* check grammar
* understand relationships between words

---

## 🔸 3. Semantics (Meaning of Words)

Semantics focuses on **meaning**.

![Image](https://images.openai.com/static-rsc-4/LqEOiZmS_UkJfk8aoDbKaTII-sSi9lGsb4GNtRw9MsPWnKPR34jkTuF2SeRADtREaMrq-QRzqgNvjN2CKlOfse9thSy0ThV7xfPn3oOgJg8my5gxp3zVHovcWnybd14i_XFFYrHu0lpL8NMvB0SuII9LMLPN7Munmq9wYIvTmUdtY7CZ5LyzTcMFhoR2GncE?purpose=fullsize)

### Example:

* “Apple is red”

👉 Meaning:

* Apple = fruit (not company)

👉 NLP uses semantics to:

* understand correct meaning
* resolve ambiguity

---

## 🔸 4. Pragmatics (Context and Intent)

Pragmatics deals with **context and intention behind words**.

![Image](https://images.openai.com/static-rsc-4/FVTULuyXxl0Gpg4REBVJpA00rqcBKzotx3n_3rmUArywuw8kV9jZBCtANiZXOTkWca9oirC6eF5TzuPhyTwO403_hP0AmQPhe-0NM2FFEHD8bOULCLiCSDRs0zKGUx_A4H2_6KRtFqvhShraf4xDaVRR8ahoVOta-WfZjeIGlLLd0RTqywZNENiGYhQRL_yq?purpose=fullsize)

### Example:

* “Can you open the door?”

👉 Literal meaning → Question
👉 Actual meaning → Request

👉 NLP uses pragmatics to:

* understand user intent
* improve chatbots and assistants

---

## 🔸 5. Phonology (Sound of Language)

Phonology studies **sounds in language**.

![Image](https://images.openai.com/static-rsc-4/3JFZ3jxuLmLGtJwjZJ2BN-wHwnNQ7AqcSkVMECrsupIPwczx0M2z0810JCC8LpIAB0iq6M_OWMsIjo6sp8bmVB8TDRYqs4GsEmFYz9vYg0wfbBhMh6gFdTcD7y13ywOo_B7cDf1mEnRweyg9oF2gprIjXcm_T92SsWOs1VElvkzN4hH9WcWSxoeEMCdpeZPy?purpose=fullsize)

### Example:

* “bat” vs “pat”
  👉 Only one sound changes meaning

👉 Used mainly in:

* Speech recognition
* Voice assistants

---

## 🔹 How These Levels Work Together

Let’s take a sentence:

> “The boy is playing football”

| Level      | What it does                                      |
| ---------- | ------------------------------------------------- |
| Morphology | Understands “playing” = play + ing                |
| Syntax     | Identifies subject, verb, object                  |
| Semantics  | Understands meaning of sentence                   |
| Pragmatics | Understands context (casual talk, sports context) |

👉 All levels together help machines understand language like humans.

---

## 🔹 A Simple Real-Life Analogy

Think of language like building a house:

* Morphology → bricks (words)
* Syntax → structure (how bricks are arranged)
* Semantics → purpose (what the house means)
* Pragmatics → usage (how people use the house)

👉 NLP builds understanding step by step, just like constructing a house.

---

## 🔹 Importance in NLP Applications

Linguistics helps in:

* **Machine Translation** → correct meaning across languages
* **Chatbots** → understanding user intent
* **Sentiment Analysis** → detecting emotions
* **Speech Systems** → recognizing spoken words

---

## 🔹 Summary

* Linguistics provides the **foundation for NLP**
* Language is analyzed at multiple levels:

  * Morphology → word structure
  * Syntax → grammar
  * Semantics → meaning
  * Pragmatics → context
  * Phonology → sound
* These levels work together to help machines understand human language
